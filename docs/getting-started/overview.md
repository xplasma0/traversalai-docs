# Overview

TraversalAI is a messaging-first AI control system that lets you run and manage AI workflows through dashboard, CLI, and channel integrations.

## Key Capabilities

- Persistent sessions and long-running workflows
- Multi-channel communication surfaces
- Configurable tools and routing
- Secure self-hosted deployment support

## Product Surfaces

- Dashboard: visual operations and chat control
- CLI: automation and scripting
- API/Integrations: external system connectivity
